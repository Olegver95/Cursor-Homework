Basic git commands

  • config --global user.name blabla - встановлює ім'я користувача
  • config --global user.email blabal@bla -встановлює електронну пошту користувача
  • git init - створює локальний репозиторій
  • git status - провірка стану локального репозиторія
  • git add - добавити файл в буфер
  • git add. - додати до відслідковування змін
  • git commit -m "commit text" - коміт. відправляємо вміст буфера в репозиторій
  • git log - перегляд історії комітів
  • git checkout - дозволяє пересуватись між комітами та гілками
  • git branch - створює, редагує, видаляє гілки
  • git push - відправляє внесені зміни на віддалений репозиторій
  • git pull - стягує зміни з віддаленого репозиторія
  • git merge name_branch - зливання бранча із текучим бранчом (в якому ми зараз знаходимося)
  • git clone - клонування віддаленого репозиторія

    • .gitignore - служить для вказівки в ньому файлів і папок, які потрібно приховати від системи контролю версій Git

    Advanced git commands

  • git revert "hash" - відміняє коміт (виконує повну протилежність, те що було знищено вертає, те що було добавлено знищує)
  • git reset - обираємо коміт і всі коміти після нього будуть видалені
  • git rebase - склеювання комітів з їх перезаписом
  • git cherry-pick - використовується для перенесення комітів з одної гілки на іншу
  • git stash - призначена для того, щоб помістити поточні зміни, які ви виконали в файлах, в окреме сховище, і повернути файли до вихідного стану
